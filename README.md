# 봉사활동 캘린더

정적 호스팅용 HTML 페이지입니다. `index.html`을 브라우저에서 열거나 OSS Pages/GitHub Pages 같은 정적 페이지 서비스에 올리면 됩니다.

## 파일

- `index.html`: 화면 구조
- `styles.css`: 깔끔한 운영 캘린더 UI
- `app.js`: 2026년 7월부터 2027년 6월까지의 일정 데이터, 날짜 선택, 공지문 생성, 입력 저장, JSON 백업

## 달력 사용

연간 달력에서 날짜를 누르면 해당 날짜의 봉사활동 상세가 열립니다. 봉사활동이 없는 날짜를 누르면 `이 날짜에 봉사 등록` 버튼이 나오고, 등록 후 담당자, 시간, 장소, 공지문 정보를 입력할 수 있습니다.

## 공지문 생성

상세 입력 영역에서 봉사명, 봉사일, 시간, 장소, 복장, 인원, 참가비, 계좌, 세부 일정을 입력하면 공지문 영역이 자동으로 갱신됩니다. `공지 복사` 버튼을 누르면 단톡방에 붙여넣을 수 있는 텍스트가 복사됩니다.

## 저장 방식

페이지에서 입력한 값은 현재 브라우저의 `localStorage`에 저장됩니다. 다른 사람에게 같은 데이터를 전달하려면 페이지 상단의 `JSON 내보내기`로 파일을 저장한 뒤, 다른 브라우저에서 `JSON 가져오기`를 사용하면 됩니다.

공개 페이지에서 여러 사람이 같은 데이터를 동시에 수정해야 한다면 다음 단계로 Google Sheets, Supabase, GitHub 저장소 커밋 자동화 중 하나를 붙이는 편이 좋습니다.
